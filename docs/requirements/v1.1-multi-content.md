## 三、赛季系统

### 3.1 数据模型

```python
@dataclass
class Season:
    """赛季"""
    season_id: str
    name: str
    start_time: datetime
    end_time: datetime
    is_active: bool
    
    # 奖励配置
    rewards: Dict[str, SeasonReward]  # tier -> reward

@dataclass
class SeasonReward:
    """赛季奖励"""
    tier: str  # gold, diamond, master, grandmaster
    avatar_frame: Optional[str]
    skin: Optional[str]
    title: Optional[str]
    gold: int
    exp: int

@dataclass
class PlayerSeasonData:
    """玩家赛季数据"""
    player_id: str
    season_id: str
    highest_tier: Tier
    final_tier: Tier
    total_games: int
    total_wins: int
    top4_rate: float
    first_place_count: int
    
    # 通行证
    pass_level: int
    pass_exp: int
    pass_premium: bool
```

### 3.2 功能要点

1. **赛季时长**: 28天
2. **段位软重置**: 新赛季初始段位 = (上赛季段位 + 青铜) / 2
3. **赛季通行证**:
   - 免费轨道: 基础奖励
   - 付费轨道: 高级奖励
   - 任务获取通行证经验
4. **赛季任务**:
   - 每日任务: 3个
   - 周任务: 5个
   - 赛季挑战: 10个

## 四、成就系统

### 4.1 数据模型

```python
@dataclass
class Achievement:
    """成就"""
    achievement_id: str
    name: str
    description: str
    category: str  # collection, battle, social, special
    tier: int  # 1=铜, 2=银, 3=金
    requirement: AchievementRequirement
    rewards: AchievementReward
    icon: str

@dataclass
class AchievementRequirement:
    """成就需求"""
    type: str  # count, collect, win_streak, damage, etc.
    target: int
    conditions: Dict[str, Any]

@dataclass
class AchievementReward:
    """成就奖励"""
    gold: int
    exp: int
    avatar_frame: Optional[str]
    title: Optional[str]
    item: Optional[str]

@dataclass
class PlayerAchievement:
    """玩家成就"""
    player_id: str
    achievement_id: str
    progress: int
    completed: bool
    completed_at: Optional[datetime]
```
