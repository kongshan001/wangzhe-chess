## 需求 #11: 皮肤系统

**优先级**: P2 | **预计工时**: 4小时

### 描述
为英雄添加皮肤系统，皮肤带有微弱属性加成和视觉效果。

### 功能要点
1. **皮肤分类**
   - 普通皮肤：纯视觉变化
   - 稀有皮肤：+5%某一属性
   - 史诗皮肤：+5%多属性 + 特效
   - 传说皮肤：+8%多属性 + 全新技能特效

2. **皮肤获取**
   - 商店购买（金币/钻石）
   - 赛季奖励
   - 成就解锁
   - 活动限定

3. **皮肤管理**
   - 皮肤收藏界面
   - 一键穿戴
   - 皮肤预览

### 验收标准
- [ ] 英雄可装备皮肤
- [ ] 皮肤属性加成生效
- [ ] 视觉特效正确显示
- [ ] 皮肤收藏记录持久化

## 需求 #12: 英雄碎片系统

**优先级**: P2 | **预计工时**: 3小时

### 描述
通过收集英雄碎片合成英雄，增加收集乐趣。

### 功能要点
1. **碎片获取**
   - 对局奖励
   - 每日任务
   - 商店购买
   - 分解重复英雄

2. **碎片合成**
   - 100碎片 = 1星英雄
   - 3个1星 + 50碎片 = 2星英雄
   - 3个2星 + 100碎片 = 3星英雄

3. **碎片背包**
   - 显示所有英雄碎片数量
   - 一键合成
   - 批量分解

### 验收标准
- [ ] 碎片正确获取
- [ ] 合成逻辑正确
- [ ] 碎片数据持久化
- [ ] 背包界面可用

## 需求 #13: 随机事件系统

**优先级**: P2 | **预计工时**: 3小时

### 描述
对局中随机触发特殊事件，增加游戏变数和趣味性。

### 功能要点
1. **事件类型**
   - 金币雨：所有玩家获得额外金币
   - 英雄折扣：商店某费用英雄打折
   - 装备掉落：野怪掉落额外装备
   - 羁绊祝福：某羁绊效果翻倍
   - 幸运轮盘：随机获得奖励

2. **触发机制**
   - 每回合5%概率触发
   - 特定回合固定触发（如10/20/30回合）
   - 可配置触发条件

3. **事件界面**
   - 事件公告动画
   - 事件详情说明
   - 事件历史记录

### 验收标准
- [ ] 事件随机触发
- [ ] 事件效果正确生效
- [ ] 事件公告显示正常
- [ ] 事件可配置
