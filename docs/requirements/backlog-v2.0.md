## 需求 #21: 公会/战队系统

**优先级**: P1 | **预计工时**: 8小时

### 功能描述
建立完整的公会系统，支持玩家组建战队，参与公会活动和对战。

### 功能要点

#### 1. 公会基础
- **创建公会**: 消耗金币/钻石创建，设置名称、图标、宣言
- **加入公会**: 搜索申请、邀请码、审批机制
- **公会等级**: 通过活跃度升级，解锁更多功能
- **公会职位**: 会长、副会长、长老、成员

#### 2. 公会功能
- **公会聊天**: 实时群聊、公告板
- **公会捐献**: 捐献金币/钻石获得贡献点
- **公会商店**: 贡献点兑换道具/皮肤
- **公会任务**: 每周集体任务，全员奖励

#### 3. 公会对战
- **公会战**: 每周公会间对战
- **领地争夺**: 占领地图资源点
- **排行榜**: 公会积分排行

### 验收标准
- [ ] 可以创建/加入/退出公会
- [ ] 公会聊天正常工作
- [ ] 公会任务和奖励正确发放
- [ ] 公会战匹配和对战逻辑正确
- [ ] 公会数据持久化

### 实现价值

#### 4. 积分系统

- **年度积分**: 年度积分兑换奖励

### 验收标准
