# 王者之奕 - 游戏设计文档

## 一、游戏概述

### 1.1 游戏简介
**王者之奕**是一款基于王者荣耀IP的8人自走棋策略对战游戏。玩家通过招募英雄、搭配羁绊、调整站位、管理经济，在回合制战斗中击败其他7位玩家，成为最后的王者。

### 1.2 游戏定位
- **类型**：策略自走棋 / Auto Battler
- **平台**：移动端（iOS/Android）
- **单局时长**：25-40分钟
- **目标用户年龄**：12+

### 1.3 核心循环
```
招募英雄 → 搭配羁绊 → 调整站位 → 战斗阶段 → 获胜/失败 → 重复
```

## 二、核心玩法

### 2.1 游戏流程

#### 2.1.1 准备阶段
- **开局**：每位玩家获得100点生命值
- **共享卡池**：所有玩家从同一英雄池中抽取英雄
- **初始金币**：第1回合获得基础金币

#### 2.1.2 回合结构
每个回合分为两个阶段：

| 阶段 | 时长 | 操作内容 |
|------|------|----------|
| **准备阶段** | 30秒 | 购买/出售英雄、调整站位、升级经验 |
| **战斗阶段** | 40秒（或战斗结束） | 自动战斗 |

#### 2.1.3 伤害计算
- **失败方**：受到基于对手剩余英雄数量和星级的伤害
- **基础伤害**：每只敌方存活英雄造成1点伤害
- **星级加成**：2星+1伤害，3星+3伤害
- **野怪回合**：失败不扣血，胜利获得装备/金币奖励

### 2.2 经济系统

#### 2.2.1 金币来源
| 来源 | 数量 | 说明 |
|------|------|------|
| 基础收入 | +1 | 每回合固定获得 |
| 胜利奖励 | +1 | 战斗胜利获得 |
| 连胜/连败 | +1~+3 | 连续2/3/4+场 |
| 利息 | +1~+5 | 每10金币+1，上限50 |
| 出售英雄 | 按星级返还 | 1星返还购买价的50% |

#### 2.2.2 金币用途
- **刷新商店**：2金币（随机刷新5个英雄）
- **购买英雄**：按费用支付（1~5金币）
- **升级经验**：4金币购买4点经验
- **购买经验**：直接升级到下一等级

### 2.3 英雄系统

#### 2.3.1 费用与概率
| 玩家等级 | 1费 | 2费 | 3费 | 4费 | 5费 |
|----------|-----|-----|-----|-----|-----|
| 1级 | 100% | 0% | 0% | 0% | 0% |
| 3级 | 70% | 30% | 0% | 0% | 0% |
| 5级 | 40% | 35% | 23% | 2% | 0% |
| 7级 | 26% | 30% | 30% | 12% | 2% |
| 9级 | 10% | 15% | 30% | 30% | 15% |

#### 2.3.2 升星机制
- **1星 → 2星**：3个相同1星英雄合成
- **2星 → 3星**：3个相同2星英雄合成
- **属性提升**：每升一星，基础属性提升约80%~100%

#### 2.3.3 等级与上阵数量
| 玩家等级 | 可上阵英雄数 | 升级所需经验 |
|----------|--------------|--------------|
| 1级 | 1 | 0 |
| 2级 | 2 | 2 |
| 3级 | 3 | 6 |
| 4级 | 4 | 10 |
| 5级 | 5 | 20 |
| 6级 | 6 | 36 |
| 7级 | 7 | 56 |
| 8级 | 8 | 80 |
| 9级 | 9 | 100 |

### 2.4 羁绊系统

#### 2.4.1 羁绊类型
- **种族羁绊**：基于英雄背景故事（如长城守卫军、稷下学院等）
- **职业羁绊**：基于英雄定位（如坦克、法师、刺客等）

#### 2.4.2 羁绊效果
- **触发条件**：上阵指定数量的同羁绊英雄
- **效果层级**：通常有2/4/6或3/6/9等不同层级
- **效果类型**：属性加成、特殊机制、光环效果

### 2.5 装备系统

#### 2.5.1 装备获取
- **野怪回合掉落**：击败野怪有概率获得
- **装备选择**：可选择一件装备或出售换取金币

#### 2.5.2 装备合成
- **基础装备**：8种基础装备
- **合成装备**：两件基础装备合成一件高级装备
- **装备限制**：每个英雄最多装备3件

## 三、目标用户

### 3.1 核心用户画像

#### 3.1.1 主要用户群体
| 特征 | 描述 |
|------|------|
| **年龄层** | 18-35岁 |
| **性别** |男性60%，女性40% |
| **职业** |学生、白领、自由职业者 |
| **游戏习惯** |日均游戏1-3小时 |
| **消费能力** |中低消费为主 |

#### 3.1.2 用户需求分析
- **策略爱好者**：享受思考和规划的乐趣
- **IP粉丝**：对王者荣耀角色有情感连接
- **休闲玩家**：碎片时间娱乐需求
- **社交需求**：与好友组队或观战

### 3.2 用户动机

#### 3.2.1 核心驱动力
1. **策略深度**：每次游戏体验不同
2. **收集养成**：解锁新英雄、皮肤
3. **竞技排名**：段位提升带来成就感
4. **社交互动**：与好友一起游戏

#### 3.2.2 留存要素
- **赛季更新**：每赛季新增/调整英雄和羁绊
- **排位系统**：从青铜到王者的段位晋升
- **成就系统**：长期目标激励
- **限时活动**：特殊玩法和奖励

## 四、差异化特色

### 4.1 IP优势

#### 4.1.1 角色认知
- **认知度高**：王者荣耀拥有庞大的玩家基础
- **情感连接**：玩家对角色有深厚的情感
- **背景故事**：丰富的世界观支持羁绊设计

#### 4.1.2 美术资源
- **3D模型复用**：可复用王者荣耀英雄模型
- **技能特效**：可直接引用或简化
- **皮肤系统**：丰富的皮肤可转换为奖励

### 4.2 玩法创新

#### 4.2.1 特色机制
| 机制 | 描述 | 差异化价值 |
|------|------|------------|
| **天赋系统** | 每3回合可选择一个天赋强化阵容 | 增加策略变化 |
| **双人模式** | 两人共享生命值，可互相传递英雄 | 增强社交属性 |
| **快速模式** | 缩短回合时间，加快节奏 | 满足碎片时间需求 |
| **挑战模式** | 每周特殊规则（如全随机英雄） | 增加新鲜感 |

#### 4.2.2 战斗特色
- **大招连招**：英雄技能有连携效果
- **地形互动**：部分棋盘有特殊地形
- **动态天气**：天气系统影响战斗

### 4.3 社交系统

#### 4.3.1 好友系统
- **好友观战**：可观看好友对战
- **阵容分享**：一键复制好友阵容
- **排行榜**：好友排名对比

#### 4.3.2 赛季活动
- **战队系统**：组队参与战队赛
- **赛季皮肤**：达到指定段位奖励
- **赛季回忆**：生成赛季数据报告

### 4.4 技术优势

#### 4.4.1 服务器架构
- **低延迟**：采用区域服务器部署
- **断线重连**：支持中途断线重连
- **观战系统**：实时观战，延迟15秒

#### 4.4.2 AI辅助
- **智能推荐**：新手阵容推荐
- **装备建议**：根据阵容推荐装备
- **羁绊提示**：显示可用羁绊组合

## 五、商业化设计

### 5.1 收入来源

#### 5.1.1 主要收入
| 类型 | 价格区间 | 说明 |
|------|----------|------|
| **通行证** | 68元/赛季 | 解锁额外奖励 |
| **小小英雄** | 6-30元 | 可交互的吉祥物 |
| **棋盘皮肤** | 18-68元 | 不同主题棋盘 |
| **英雄皮肤** | 6-28元 | 英雄3星时展示 |

#### 5.1.2 非付费内容
- **免费通行证**：基础奖励
- **活动获取**：通过活动获得皮肤
- **金币购买**：部分内容可用金币购买

### 5.2 平衡性保证

#### 5.2.1 公平竞技
- **无付费优势**：不售卖影响平衡的道具
- **统一卡池**：所有玩家使用相同英雄池
- **数据驱动**：基于数据调整平衡

#### 5.2.2 更新节奏
- **热更新**：每周调整平衡性问题
- **版本更新**：每月新增内容
- **赛季更新**：每3个月大版本更新

## 六、风险管理

### 6.1 技术风险
- **服务器压力**：高峰期并发问题
- **作弊防护**：防止外挂和脚本
- **数据安全**：保护玩家数据

### 6.2 运营风险
- **用户流失**：保持新鲜感和平衡性
- **负面舆论**：及时响应玩家反馈
- **竞品压力**：持续创新和优化

### 6.3 应对策略
- **灰度发布**：新功能小范围测试
- **玩家调研**：定期收集玩家反馈
- **数据分析**：基于数据做决策

## 七、成功指标

### 7.1 核心KPI
| 指标 | 目标值 | 说明 |
|------|--------|------|
| **DAU** | 100万+ | 日活跃用户 |
| **留存率(次日)** | 40%+ | 次日回访率 |
| **留存率(7日)** | 20%+ | 7日回访率 |
| **平均游戏时长** | 30分钟/日 | 人均日游戏时长 |
| **付费率** | 5%+ | 付费用户占比 |
| **ARPU** | 15元+ | 每用户平均收入 |

### 7.2 健康指标
- **平衡性**：阵容多样性（前10阵容胜率差<5%）
- **对局时长**：25-40分钟分布占比>80%
- **新手转化**：完成3局游戏的玩家>70%
