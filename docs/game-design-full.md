# 王者之奕 - 游戏设计文档

> 版本: 0.1.0 | 更新日期: 2026-02-21

## 一、游戏概述

### 1.1 游戏简介

王者之奕是一款 8 人对战的自动战斗棋类游戏（Auto Chess）。玩家通过购买英雄、搭配羁绊、排兵布阵，在自动战斗中击败对手，成为最后的赢家。

### 1.2 目标用户

- 策略游戏爱好者
- MOBA 玩家（熟悉英雄设定）
- 休闲竞技玩家
- 15-35 岁年龄段为主

### 1.3 核心体验

- **策略深度**: 羁绊组合、阵容搭配
- **随机乐趣**: 英雄抽取、商店刷新
- **快节奏**: 单局 15-20 分钟
- **社交属性**: 排位竞技、好友对战

## 二、核心玩法

### 2.1 游戏流程

```
准备阶段 (30秒) → 战斗阶段 (自动) → 结算阶段 → 循环
```

| 阶段 | 时长 | 玩家操作 |
|------|------|----------|
| 准备 | 30秒 | 购买/出售英雄、部署阵容、升级 |
| 战斗 | 自动 | 观看战斗 |
| 结算 | 3秒 | 计算伤害、发放奖励 |

### 2.2 胜负判定

- 玩家初始 100 生命值
- 战斗失败扣血，扣血量 = 存活敌方英雄 × 2
- 生命值归零则淘汰
- 最后存活的玩家获胜

## 六、推荐阵容

## 七、数值平衡原则

### 7.1 设计原则

1. **无绝对最优解**: 每种阵容都有克制
2. **滚雪球控制**: 连败奖励帮助翻盘
3. **随机与策略**: 运气影响但有策略空间
4. **快节奏**: 单局控制在15-20分钟

### 7.2 平衡指标

| 指标 | 目标值 |
|------|--------|
| 平均对局时长 | 15-20分钟 |
| 首位淘汰回合 | 8-12回合 |
| 阵容多样性 | >10种主流阵容 |
| 英雄出场率 | 标准差 <15% |
