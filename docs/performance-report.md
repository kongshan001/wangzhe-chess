## 4. 优化实现

### 4.1 英雄池操作优化

**文件**: `src/server/perf/hero_pool_optimized.py`

优化策略：
- 按费用分桶存储，减少搜索范围
- 预计算累积概率表，快速随机抽取
- 批量操作减少锁竞争
- 支持 numpy 加速（可选）

### 4.2 羁绊计算优化

**文件**: `src/server/perf/synergy_cache.py`

优化策略：
- LRU 缓存羁绊计算结果
- 增量更新只计算变化部分
- 状态哈希快速判断是否需要重新计算

### 4.3 WebSocket 消息优化

**文件**: `src/server/perf/ws_batch.py`

优化策略：
- 消息聚合减少发送次数
- 批量发送提高吞吐
- 优先级队列保证重要消息优先
- 背压控制防止内存溢出

### 4.4 数据库索引优化

**文件**: `migrations/versions/perf_indexes_001.py`

添加的索引：
- `ix_players_active_login` - 活跃玩家查询
- `ix_player_ranks_leaderboard` - 排行榜查询
- `ix_player_stats_winrate` - 胜率统计
- `ix_custom_room_status` - 房间状态查询
