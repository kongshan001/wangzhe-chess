## 🔴 高优先级问题

### H-001: 装备系统 - 类型安全
- **位置**: src/server/game/equipment.py
- **描述**: EquipmentStats 和 Equipment 缺少输入验证
- **风险**: 恶意JSON可注入任意数据
- **建议**: 使用 Pydantic 模型替代 dataclass
```python
from pydantic import BaseModel

class EquipmentStats(BaseModel):
    attack: int = 0
    armor: int = 0
    # 自动验证
```

### H-002: 天赋系统 - 性能问题
- **位置**: src/server/game/talent.py
- **描述**: apply_to_player 遍历所有英雄，O(n*m)复杂度
- **建议**: 使用缓存避免重复计算

### H-003: 房间管理 - 线程安全
- **位置**: src/server/room/game_room.py
- **描述**: 房间状态修改未加锁
- **风险**: 并发访问可能导致竞态条件
- **建议**: 使用 asyncio.Lock 保护共享状态
