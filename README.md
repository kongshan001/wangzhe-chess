# 王者之奕 - 自走棋游戏

> 生产级自走棋游戏 Python 实现，支持 8 人在线对战

## 🎮 项目简介

本项目是一个从零开始实现的自走棋（Auto Chess）游戏，灵感来自王者荣耀的王者模拟战模式。目标是实现一个生产级别的、可部署的自走棋游戏服务器。

### 核心特性

- 🎯 **确定性战斗模拟** - 保证相同输入产生相同输出，支持回放
- 🔄 **帧同步架构** - 混合帧同步与状态同步，优化网络传输
- 🏠 **房间管理** - 支持 8 人房间、匹配系统
- 🎭 **羁绊系统** - 完整的种族/职业羁绊计算
- 💰 **经济系统** - 连胜/连败奖励、利息机制
- 📦 **共享英雄池** - 8 人共享英雄池抽取机制

## 🚀 快速开始

### 安装依赖

```bash
# 使用 uv (推荐)
uv pip install -e ".[dev]"

# 或使用 pip
pip install -e ".[dev]"
```

### 运行测试

```bash
pytest tests/ -v
```

### 启动服务器

```bash
# 开发模式
uvicorn src.server.main:app --reload --port 8000

# 生产模式
wangzhe-server
```

## 🎯 游戏机制

### 英雄系统

- **星级**: 1-3 星，3 个同名低星英雄可合成 1 个高星英雄
- **费用**: 1-5 金币，影响购买价格和刷新概率
- **属性**: HP、攻击力、防御力、攻速、技能

### 羁绊系统

- **种族羁绊**: 人族、神族、魔种、亡灵、精灵、兽族、机械、龙族、妖精
- **职业羁绊**: 战士、法师、刺客、射手、坦克、辅助、游侠、术士
- **效果**: 属性加成、特殊技能、被动效果

### 经济系统

- **基础收入**: 每回合 5 金币
- **利息**: 每 10 金币获得 1 利息（最高 5 金币）
- **连胜/连败奖励**: 最高 +3 金币
- **刷新商店**: 2 金币
- **购买经验**: 4 金币 = 4 经验
